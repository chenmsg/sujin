•注册

•填写本人手机号

•设置6到16位的密码

•邀请码填写邀请人得手机号，必须要填写哦！

•登录

•登录是APP的唯一入口，不注册是看不到APP的内容滴！

•实名认证

•从我的→基本信息或者收款→选择支付卡都可以进去实名认证

•填写自己真实姓名和身份证号

•按照要求上传照片，这是为了用户资金安全，确保是本人在使用

•绑定结算卡

•入口同上

•填写您本人的结算卡（储蓄卡），作为收款用

•按照要求上传照片，这是为了用户资金安全，确保是本人在使用

•绑定支付卡

•如实填写信用卡的各种信息，如果信息有误会导致刷卡不成功

•选择通道

•无积分

•实时到账，时间段有限制

•无积分顾名思义就是刷卡不会得到积分

•优势是费用低，成本越低，利润越大

•劣势:有些朋友可能需要积分兑换东西，很多银行积分活动不错，我会在用卡攻略里面介绍

•有积分

•实时到账，没有时间限制，深夜或者凌晨刷卡的话，可能到账有延迟

•优势:有积分，很多银行积分可以兑换很多物品，生日期间的刷卡，很多银行是4倍积分，这时候可以薅一笔

•劣势:就是价格贵呗

•开通快捷支付

•刷卡支付的时候会自动判断当前所选银行卡是否已开通快捷支付，开通后无需再次开通

•快捷支付就是手机端通过验证码即可进行交易

•开通快捷交易是必须滴，不然没法交易

•收款

•看清需要收款的金额

•输入获取到的验证码

•显示收款成功，并收到银行的扣款提示，等待money到账即可

•查询交易记录

•入口:收款页面右上角

•里面有用户每一条的交易，如果交易出现迟迟不到账的情况，提供交易列表的订单号，可以让我们的运行人员快速查询问题所在

•常见问题

•收款成功，却没收到钱？

•首先确认有没有信用卡有没有扣款成功，如果成功却没有到账，建议先等等，如果超过了一个小时还没到，再来联系我们的运营人员即可

•提供交易记录里面的订单号和银行卡的扣款成功短信，让运营人员查询

•通道没有额度？

•现在无积分通道资源紧张，所以大家最好在通道一开放就开始用，晚了就容易没额度
